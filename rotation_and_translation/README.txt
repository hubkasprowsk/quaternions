To use this script user needs to provide command for running python script with name of input xyz file molecule, name of output xyz file and three values which in case of rotations are angles of rotations around x, y and z axis, and in case of translation are components of a translation vector.

Examples:

Rotation

python rotate.py molecule.xyz molecule_rotated.xyz 90 90 90

Translation

python translate.py molecule.xyz molecule_translated.xyz 3 4 5
